# lighthouse-puppeteer

## Preprquisite:
#  Visual studio Code
# npm latest version

#Steps:
 1. Clone to local drive
 2. open project root folder in visual studio code
 3. open terminal
 4. type npm install
 5. hit enter
 6. enter npn test
 7. test will start executing
